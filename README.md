# CANALI-IPTV
Here you can find all the digital terrestrial channels including Rai and Mediaset
Questa repository contiene tutti i principali canali TV e Radio nazionali italiani in un file m3u da aprire con un qualsiasi programma che supporti file playlist. I link sono tutti estrapolati dagli stream ufficiali, e per questo dovrebbero essere visibili legalmente soltanto all'interno dell'Italia. Intendo aggiornare i link non funzionanti o aggiungere nuovi canali se vedo commenti che lo chiedono ed è nelle mie capacità farlo legalmente.
